Task Management v0.0.1

Routers:

Indice Method Route Description
1 GET / Apresenta a página inicial do sistema
2 GET /api Apresenta a página de dumentação da API
3 GET /api/tasks Apresenta a lista de tarefas
4 POST /api/tasks Rota para adicionar novas tarefas
5 DELETE /api/tasks/:id Rota para apagar uma tarefa
6 PUT /api/tasks/:id Rota para actualizar uma tarefa
